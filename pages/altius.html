Altius Setup

Altius is our local-only agent framework for civic learning and chapter support. This website hosts docs and downloads. The agent runtime (backend) runs on your computer, not on GitHub Pages.

Static site + local backend
Download Altius Starter Pack Open v1.2.0 Release View /altius folder
What you’re downloading: a starter bundle from the repo’s /altius/ folder. It includes scaffolding, example config, and docs. It does not contain secrets.

Install (local)

GitHub Pages can’t run Python servers. So the site stays static and the agent runs locally.

  1. Download the Starter Pack zip and unzip it.
  2. Open the folder and read: altius/README.md
  3. Create your local config:
    • Copy altius/config.example.json → altius/config.json
    • Fill in required values (keep config.json local only)
  4. When you standardize the runtime (FastAPI/Ollama/etc.), follow the instructions in altius/README.md.

Collaboration rule: commit scaffold + docs only. Don’t commit secrets, tokens, or local indexes.

How this works

GitHub Pages serves static files. Altius needs a runtime (backend) to do anything agent-like.

  • This site: public docs, links, downloads
  • Your computer: the agent runtime + local indexing

Troubleshooting

  • Download link fails: If the download fails, open the release page and download the zip there.
  • Nothing runs when opened: correct. The zip is scaffolding. You still need a local backend runtime.
  • Port already in use: if/when you run a local server, close other apps using that port or change the port in local config.
  • Missing Python/Ollama: install locally per altius/README.md.
