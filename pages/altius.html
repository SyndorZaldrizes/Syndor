Altius Setup

Altius is our local-only agent framework for civic learning and chapter support. This website hosts docs and downloads. The agent runtime (backend) runs on your computer, not on GitHub Pages.

Download Altius Starter Pack View /altius folder
What you’re downloading: a starter bundle from the repo’s /altius/ folder. It includes scaffolding, example config, and docs. It does not contain secrets.

Install (local)

This is intentionally simple. If you’re trying to run Altius on GitHub Pages, you’re going to have a bad time (Pages can’t run Python servers).

  1. Download the Starter Pack zip and unzip it.
  2. Open the folder and read: altius/README.md
  3. Create your local config:
    • Copy altius/config.example.json → altius/config.json
    • Fill in any required values (keep config.json local only)
  4. If/when you add the backend runtime locally (FastAPI/Ollama/etc.), follow the instructions in altius/README.md. (This page avoids hardcoding commands because your local setup may differ.)

If you’re collaborating, only commit changes to the scaffold and docs. Do not commit secrets, tokens, or local indices.

How this works

GitHub Pages serves static files. Altius needs a runtime (backend) to do anything “agent-like.” So we split responsibilities:

  • This site: public docs, links, and downloads
  • Your computer: the agent runtime and any local indexing

Troubleshooting

  • Download link 404: confirm altius-starter.zip is attached under the v1.2.0 Release “Assets” section. If needed, re-upload the zip to that release.
  • Nothing “runs” when opened: correct. The zip is scaffolding. You still need to run the backend locally to use an agent.
  • Port already in use: if/when you run a local server, close other apps using that port or change the port in your local config.
  • Missing Python/Ollama: install locally per the instructions in altius/README.md once you standardize the runtime setup.
