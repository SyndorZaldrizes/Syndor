# Code of Conduct

This repository hosts the website for the **Maryland Student Legislature – St. Mary’s College of Maryland Chapter (SMCM MSL)**.

We expect everyone interacting with this project (maintainers, contributors, and visitors) to:

- Be respectful and professional in all communication.
- Avoid harassment, discrimination, and personal attacks of any kind.
- Assume good faith, ask questions instead of making accusations.
- Keep discussions focused on improving the site and chapter resources.

## Unacceptable Behavior

Examples of behavior that is not welcome:

- Harassment, hate speech, or slurs.
- Threats or encouragement of violence.
- Persistent trolling or derailing conversations.
- Sharing private or personal information without consent.

## Scope

This Code of Conduct applies to:

- This GitHub repository (issues, pull requests, comments).
- Any related communication channels used to discuss this project.

## Enforcement

Project maintainers may:

- Edit, lock, or remove issues and comments that violate this Code of Conduct.
- Block or report users who engage in repeated or severe violations.

Maintainers will review reports and take appropriate action as time and capacity allow.
